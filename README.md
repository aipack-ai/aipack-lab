# AIPACK Lab - Project Examples

Here is a list of AIPACK lab packs that can be ran from this repo. 

First, make sure to install the latest .aipack 
For example:

```sh
cd hello-world
aip run hello-world
```

NOTE: Make sure you have AIPACK Version `aip -V` `=> 0.7.11`

```sh
$ aip -V
aipack 0.7.13
```
